# DekstopLoginApp

Project sederhana buat belajar bikin aplikasi desktop pakai C#.  
Di sini saya coba bikin form login dengan username & password dan validasi basic.

## Fitur yang udah jadi
- Input Username & Password
- Tombol Login
- Munculin pesan kalau salah / benar
- UI simple gaya WinForms

## Tools yang dipakai
- C# (.NET)
- Windows Forms
- Visual Studio Community 2022

## Cara jalanin
1. Clone repo:
2. Buka file `.slnx` di Visual Studio
3. Tekan **F5** buat ngejalanin

## Tujuan project ini
- Belajar dasar WinForms dan event handling
- Nyoba bikin aplikasi desktop pertama
